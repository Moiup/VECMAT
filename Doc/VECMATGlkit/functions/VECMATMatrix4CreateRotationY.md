# VECMATMatrix4CreateRotationY(\_,\_)
Create the 4x4 rotation matrix around the Y axis.

## **Declaration**
```C
void VECMATMatrix4CreateRotationY(
		VECMATMatrix4 m,
		float angle
	)
```


## **Parameters**
**m**  
A 4x4 matrix.

**angle**  
An angle in radian.

## **Result**
The 4x4 rotation matrix in `m`.
