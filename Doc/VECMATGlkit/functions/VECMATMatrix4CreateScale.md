# VECMATMatrix4CreateScale(\_,\_,\_,\_,\_)
Create the 4x4 scaling matrix.

## **Declaration**
```C
void VECMATMatrix4CreateScale(
		VECMATMatrix4 m,
		float Sx,
		float Sy,
		float Sz
	)
```


## **Parameters**
**m**  
A 4x4 matrix.

**Sx**  
The X scale value.

**Sy**  
The Y scale value.

**Sz**  
The Z scale value.

## **Result**
The 4x4 scale matrix in `m`
