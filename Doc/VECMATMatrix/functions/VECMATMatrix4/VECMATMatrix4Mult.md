# VECMATMatrix4Mult(\_,\_,\_)
Do the multiplication of two matrix.

## **Declaration**
```C
void VECMATMatrix4Mult(
		VECMATMatrix4 m1,
		VECMATMatrix4 m2,
		VECMATMatrix4 res
	)
```


## **Parameters**
**m1**  
The multiplier matrix.

**m2**  
The multiplicand matrix.

**res**  
The resulting matrix.


## **Result**
The 4x4 matrix `res` with the result of `m1 * m2`.
