# VECMATMatrix4Zero(\_)
Set all values of the matrix to 0.

## **Declaration**
```C
void VECMATMatrix4Zero(VECMATMatrix4 m)
```


## **Parameter**
**m**  
The 4x4 matrix to set its values.


## **Result**
The 4x4 matrix `m` with all its values set to 0.
