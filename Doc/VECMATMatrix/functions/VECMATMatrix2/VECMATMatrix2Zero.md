# VECMATMatrix2Zero(\_)
Set all values of the matrix to 0.

## **Declaration**
```C
void VECMATMatrix2Zero(VECMATMatrix2 m)
```


## **Parameter**
**m**  
A 2x2 matrix.


## **Result**
The 2x2 matrix `m` with all its values set to 0.
