# VECMATMatrix2Identity(\_)
Set the matrix as an identity matrix.

## **Declaration**
```C
void VECMATMatrix2Identity(VECMATMatrix2 m)
```


## **Parameter**
**m**  
A 2x2 matrix.


## **Result**
The 2x2 matrix `m` as an identity matrix.
