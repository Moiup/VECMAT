# VECMATMatrix3Zero(\_)
Set all values of the matrix to 0.

## **Declaration**
```C
void VECMATMatrix3Zero(VECMATMatrix3 m)
```


## **Parameter**
**m**  
A 3x3 matrix.


## **Result**
The 3x3 matrix `m` with all its values set to 0.
