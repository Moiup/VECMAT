# VECMATMatrix3Identity(\_)
Set the matrix as an identity matrix.

## **Declaration**
```C
void VECMATMatrix3Identity(VECMATMatrix3 m)
```


## **Parameter**
**m**  
A 3x3 matrix.


## **Result**
The 3x3 matrix `m` as an identity matrix.
