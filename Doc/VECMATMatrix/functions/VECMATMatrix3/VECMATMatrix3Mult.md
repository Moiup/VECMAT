# VECMATMatrix3Mult(\_,\_,\_)
Do the multiplication of two matrix.

## **Declaration**
```C
void VECMATMatrix3Mult(
		VECMATMatrix3 m1,
		VECMATMatrix3 m2,
		VECMATMatrix3 res
	)
```


## **Parameters**
**m1**  
The 3x3 multiplier matrix.

**m2**  
The 3x3 multiplicand matrix.

**res**  
The resulting 3x3 matrix.


## **Result**
The 3x3 matrix `res` with the result of `m1 * m2`.
