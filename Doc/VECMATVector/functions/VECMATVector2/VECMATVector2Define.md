# VECMATVector2Define(_,\_,\_)
Set all values of the vector to 0.

## **Declaration**
```C
void VECMATVector2Define(VECMATVector2 v, float x, float y)
```


## **Parameters**
**v**\
The vector of size 2 to work with.

**x**\
The new `x` value.

**y**\
Tne new `y` value.

## **Result**
The vector `v` with the value `x` and `y`.
