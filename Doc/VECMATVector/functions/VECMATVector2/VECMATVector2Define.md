# VECMATVector2Define(_,\_,\_)
Set all values of the vector to 0.

## **Declaration**
```C
void VECMATVector2Zero(VECMATVector2 V)
```


## **Parameter**
**v**\
The vector of size 2 to work width.

## **Result**
All the values of `v` will be 0.
