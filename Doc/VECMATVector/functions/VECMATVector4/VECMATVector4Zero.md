# VECMATVector4Zero(_)
Set all values of the vector to 0.

## **Declaration**
```C
void VECMATVector4Zero(VECMATVector4 v)
```


## **Parameter**
**v**  
The vector of size 4 to work with.

## **Result**
All the values of `v` set to 0.
