# VECMATVector4Define(\_,\_,\_,\_,\_)
Set all values of the vector to 0.

## **Declaration**
```C
void VECMATVector3Define(
        VECMATVector4 v,
        float x,
        float y,
        float z
    )
```

## **Parameters**
**v**\
The vector of size 3 to work with.

**x**\
The new `x` value.

**y**\
Tne new `y` value.

**z**\
The new `z` value.

## **Result**
The vector `v` with the value `x`, `y` and `w`.
