# VECMATVector3DotProduct(\_,\_)
Do the dot product between two vectors.

## **Declaration**
```C
void VECMATVector3DotProduct(
        VECMATVector3 v1,
        VECMATVector3 v2
    )
```


## **Parameter**
**v1**
A vector.

**v2**
A vector.

## **Result**
Return the dot product between `v1` and `v2`.
