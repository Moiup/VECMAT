# VECMATVector3Zero(_)
Set all values of the vector to 0.

## **Declaration**
```C
void VECMATVector3Zero(VECMATVector3 v)
```


## **Parameter**
**v**  
The vector of size 3 to work with.

## **Result**
All the values of `v` set to 0.
